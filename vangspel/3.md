# Beweeg de 'vanger'

Kies een vanger, zoals een kom (Bowl in het engels).

![Een bruine kom.](../img/vangspel-3.1.png)

Sleep de kom naar de onderkant van het scherm.

Voeg deze code toe:

<pre class="blocks">
wanneer groene vlag wordt aangeklikt
herhaal
  als < toets (pijltje rechts v) ingedrukt?> dan
    verander x met (10)
  einde
  als < toets (pijltje links v) ingedrukt?> dan
    verander x met (-10)
</pre>

De pijltjes toetsen kan je uit het menu kiezen.

Druk op de pijltjes toetsen zodat de vanger naar links en rechts beweegt.

![De kom is onder de rode appel](../img/vangspel-3.2.png)
