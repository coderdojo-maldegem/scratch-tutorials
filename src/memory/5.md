# Nummertje trekken

![Dialoog met de titel nieuwe variabele met in het tekstvak de variabele naam mijn kaart nummer. Daaronder is de optie alleen voor deze sprite ingeschakeld.](../img/memory-5.png)

Klik op *maak een variabele*, kies *alleen voor deze sprite* (heel belangrijk!) en noem de variabele *mijn kaart nummer*.

Pas de code van *MaakKaarten* aan om de variabelen te gebruiken.

<pre class="blocks">
definieer (MaakKaarten :: stack) :: custom hat
maak [MijnKaartNummer v] (0)
verwijder alle van [kostuums v]
ga naar x: (-230) y: (120)
verander uiterlijk naar [achterkant]
maak grootte (50) %
verschijn
herhaal (4)
  maak x (-190)
  herhaal (6)
    volgend uiterlijk
    als <(uiterlijk [naam v]) = (achterkant)> dan
      volgend uiterlijk
    einde
    voeg (uiterlijk [naam v]) toe aan [kostuum v]
    verander [MijnKaartNummer v] met (1)
    maak een kloon van (mezelf v)
    verander x met (60)
  einde
  verander y met (-70)
einde
verdwijn
</pre>

En maak een nieuw stuk code:

<pre class="blocks">
wanneer op deze sprite wordt geklikt
als <(uiterlijk [naam v]) = (achterkant)> dan
  verander uiterlijk naar (item (MijnKaartNummer) van [kostuums v])
anders
  verander uiterlijk naar (achterkant v)
</pre>
