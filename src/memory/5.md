# Nummertje trekken

Klik op *maak een variabele*, kies *alleen voor deze sprite* (heel belangrijk!) en noem de variabele *MijnKaartNummer*.

<div class="variabele">
  <p>MijnKaartNummer</p>
  <label><input name="varscope" type="radio"/><span>Voor alle sprites</span></label>
  <label><input name="varscope" type="radio" CHECKED=""/><span>Alleen voor deze sprite</span></label>
</div>

Pas de code van *MaakKaarten* aan om de variabelen te gebruiken.

```scratch
definieer (MaakKaarten :: stack) :: custom hat
maak [MijnKaartNummer v] (0)
verwijder alle van [uiterlijken v]
ga naar x: (-230) y: (120)
verander uiterlijk naar [achterkant]
maak grootte (50) %
verschijn
herhaal (4)
  maak x (-190)
  herhaal (6)
    volgend uiterlijk
    als <(uiterlijk [naam v]) = (achterkant)> dan
      volgend uiterlijk
    einde
    voeg (uiterlijk [naam v]) toe aan [uiterlijk v]
    verander [MijnKaartNummer v] met (1)
    maak een kloon van (mezelf v)
    verander x met (60)
  einde
  verander y met (-70)
einde
verdwijn
```

En maak een nieuw stuk code:

```scratch
wanneer op deze sprite wordt geklikt
als <(uiterlijk [naam v]) = (achterkant)> dan
  verander uiterlijk naar (item (MijnKaartNummer) van [uiterlijken v])
anders
  verander uiterlijk naar (achterkant v)
```
