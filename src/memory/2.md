# Clonen

Zet deze code op de sprite van de kaart.

<pre class="blocks">
wanneer groene vlag wordt aangeklikt
ga naar x: (-230) y: (120)
verander uiterlijk naar [achterkant]
maak grootte (50) %
verschijn
herhaal (4)
  maak x (-190)
  herhaal (6)
    verander x met (1)
    maak een kloon van (mezelf v)
  einde
  verander y met (-70)
einde
verdwijn
</pre>

Onze enige kaart gaat nu rijen en kolommen aflopen, en telkens een kopietje maken.

<p class="testen">
Start je programma.
Krijg je 24 kaarten te zien?
</p>

# Symbolen

Nu willen we van elk kostuum exact 2 identieke exemplaren.
Daarvoor moeten we wat speciale truukjes toepassen.

<pre class="blocks">
volgend uiterlijk
als <(uiterlijk [naam v]) = (achterkant)> dan
  volgend uiterlijk
einde
maak een kloon van (mezelf v)
</pre>

Vervang het clone blokje in het vorige script door deze blokjes, die telkens het kostuum veranderen.
