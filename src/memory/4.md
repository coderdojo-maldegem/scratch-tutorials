# Verstoppertje

De kaarten verstoppen is heel eenvoudig met een signaal:

```scratch
wanneer groene vlag wordt aangeklikt
MaakKaarten :: custom
zend signaal (verstoppen v)
```

```scratch
wanneer ik signaal [verstoppen v] ontvang
wacht (1) sec.
verander uiterlijk naar (achterkant v)
```

Waarom een signaal en niet gewoon na de groene vlag het uiterlijk veranderen?

> Omdat alle clones van uiterlijk moeten veranderen!
