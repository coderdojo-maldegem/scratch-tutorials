# Oh oh! geraakt!

Nu moeten we er nog voor zorgen dat het personage opnieuw van bij de start moet beginnen zodra hij geraakt wordt door de bal.

Dit zijn de stappen voor je code:

1. **als** de bal je personage raakt **dan**: Zend signaal *geraakt*.
2. Wanneer het personage het signaal *geraakt* ontvangt: Richt naar 90 graden en ga naar `x: -210` en `y: -120`.

Je code zou er ongeveer zo moeten gaan uitzien:

bal:

```scratch
wanneer ik als kloon start
herhaal
  als < raak ik (Pico Walking v)?> dan
    zend signaal (geraakt v)
```

personage:

```scratch
wanneer ik signaal [geraakt v] ontvang
richt naar (90) graden
ga naar x: (-210) y: (-120)
maak [Springhoogte v] (0)
```
