# Letters verstoppen

Bij het script van de letter sprite komen enkele blokjes bij om het *streep* uiterlijk te kiezen.

Als we de grootte op 50% zetten kunnen we extra lange woorden kiezen!

```scratch
wanneer groene vlag wordt aangeklikt
richt naar (90) graden
verander uiterlijk naar [Block-streep v]
maak grootte (50) %
verdwijn
maak [woord v] (item (willekeurig getal tussen (1) en (lengte van [woorden v])) van [woorden v]
ga naar x: (-200) y: (100)
maak [i v] (0)
herhaal (lengte van (woord))
  verander [i v] met (1)
  maak [mijnletter v] (letter (i) van (woord))
  maak een kloon van (mezelf v)
  verander x met (30)
einde
```

<pre class="blocks doorhalen">
wanneer ik als kloon start
verander uiterlijk naar (voeg [Block-] en (mijnletter) samen
verschijn
```

```scratch
- wanneer ik als kloon start
- verander uiterlijk naar (voeg [Block-] en (mijnletter) samen
- verschijn
```

Bij het kloon script van de letter sprite mag het blokje dat de juiste letter laat zien weg.
We zullen later nog een extra script maken.

```scratch
wanneer ik signaal [toonletter v] ontvang
als <(mijnletter) = (antwoord)> dan
  verander uiterlijk naar (voeg [Block-] en (mijnletter) samen)
  maak [gevonden v] (1)
```

Dit script kan je nu nog niet testen. Maak eerst de variabele *gevonden* aan, voor alle sprites
