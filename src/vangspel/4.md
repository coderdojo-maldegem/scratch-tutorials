# Vang het!

![De appel sprite is geselecteerd](../img/vangspel-4.png)

Selecteer opnieuw de appel en voeg deze code toe:

```scratch
wanneer groene vlag wordt aangeklikt
herhaal
  als < raak ik (Bowl v)?> dan
    start geluid (Pop v) en wacht
    ga naar (willekeurige positie v)
    maak y (180)
```

> Klik op het Geluiden tabblad als je een ander geluid wil toevoegen, of kies een geluid uit de geluidenbibliotheek.
