# Gewonnen!

Klik op de verfborstel om een eigen sprite te tekenen.

Gebruik de **Text** knop om een boodschap te schrijven zoals *Gewonnen*.
Je kan de kleur van de letter, de grootte en de stijl veranderen.

![Het uiterlijk van de eigen sprite wordt bewerkt. De paarse tekst gewonnen is in hoofdletters.](../img/vangspel-7.png)

Met deze code kunnen we het spel beëindigen als je genoeg punten (bijvoorbeeld `20`) hebt.

```scratch
wanneer groene vlag wordt aangeklikt
verdwijn
wacht tot <(score) = (20)>
verschijn
stop [alle v]
```

Wanneer je genoeg punten scoort, krijg je nu een overwinningsboodschap!

<p class="testen">Klik op de groene vlag om te starten Speel tot je genoeg punten hebt gescoord zodat je wint!</p>
