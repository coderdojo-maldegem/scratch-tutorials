# aliens maken met klonen

## Opdracht:

- maak meerdere aliens met klonen
- gebruik slechts één sprite per alien type
- zet aliens netjes naast elkaar
- hou bij hoeveel aliens er zijn

Nieuwe blokken:

<code class="b">maak een kloon van [mijzelf v]</code>
<code class="b">wanneer ik als kloon start</code>
<code class="b">verander [ v] met ( )</code>
<code class="b">herhaal ( )</code>

<p class="testen">Test je programma!</p>

## Oplossing

<details><summary>klik hier</summary>

Achtergrond

```scratch
wanneer groene vlag wordt aangeklikt
maak [maxAantalAliens v] [9]
maak [afstandTussenAliens v] [40]
maak [aantalAliens1 v] [0]
maak [aantalAliens2 v] [0]
maak [aantalAliens3 v] [0]
maak [einde v] [0]
herhaal (maxAantalAliens)
  maak een kloon van [alien1 v]
  maak een kloon van [alien2 v]
  maak een kloon van [alien3 v]
einde
```
