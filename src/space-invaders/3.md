# aliens laten bewegen

## Opdracht:

- laat aliens bewegen met een bericht
- laat hun kostuums wisselen
- stop alles wanneer het spel gedaan is

Nieuwe blokken:

<code class="b">zend signaal [ v]</code>
<code class="b">wanneer ik signaal [ v] ontvang</code>
<code class="b">volgend uiterlijk</code>
<code class="b">herhaal tot < ></code>

Berichten laten sprites tegelijk reageren.
