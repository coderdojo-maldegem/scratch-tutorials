# Game over

Maak een sprite *game over*.

Het spel stopt wanneer de slang de rand raakt of zichzelf. De eerste is relatief eenvoudig te programmeren door onderstaande code toe te voegen aan het slangenhoofd...

```scratch
wanneer groene vlag wordt aangeklikt
herhaal
  als < raak ik (rand v) ?> dan
    zend signaal (Game Over v)
    stop [alle v]
  einde
einde
```

... en onderstaande code toe te voegen aan de sprite *game over*:

```scratch
wanneer groene vlag wordt aangeklikt
verdwijn

wanneer ik signaal [Game Over v] ontvang
verschijn
```

Voor het tweede deel van de oefening, namelijk game over wanneer de slang zijn eigen lijf raakt, is wat creativiteit en inzicht vereist!
