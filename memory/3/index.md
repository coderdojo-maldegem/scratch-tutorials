# Zelfgemaakt

Dat was het aanmaken van alle kaarten.
Laten we dit script in een apart blokje steken.
Klik op "Maak een blok".

# Snellerrrrrr!

Noem je blokje "MaakKaarten".
Om sneller te gaan, kan je ook "Voer uit zonder het scherm te verversen" aanvinken.
Maak je code los van de groene vlag en verhuis het naar het roze "Definieer MaakKaarten" blokje.

Aan de groene vlag hang je nu je nieuwe blokje.
